Image Search API

A microservice to return a JSON array of search results from Imgur.

To use, direct your browser to https://imagesearch-marshallaf.herokuapp.com/, followed by the search term. The response will be a JSON array of Imgur links and their description.

Visiting https://imagesearch-marshallaf.herokuapp.com/recent will show the 10 most recent searches.

example usage
// search for image
https://imagesearch-marshallaf.herokuapp.com/eggs%20benedict

// response
[{
url: http://imgur.com/a/Cd1xy,
text: "Eggs Benedict with crushed avocados"
}, ...
